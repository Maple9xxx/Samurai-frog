# Samurai Frog

2D side-scrolling action platformer for mobile web and PC.

## Features
- HTML5 Canvas 2D
- Vanilla JavaScript ES6 modules
- Procedural sprites and background
- Web Audio API sound effects and music
- Touch and keyboard controls
- Offline-friendly static build

## Run locally
Open `index.html` in a modern browser or serve the folder with a static HTTP server.

## Deploy
Upload the repository to GitHub Pages using the root branch folder.

## Controls
- Move: Left / Right Arrow
- Jump: Space
- Attack: Z or J
- Mobile: touch controls on screen
